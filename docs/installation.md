FastAPI-RESTful-Extension has the following dependencies:

- FastAPI <= `0.78.0`
- Python version: `3.6, 3.7, 3.8, 3.9` or `3.10`

---

### From pypi.org
Install package with pip from `pypi.org`

```console
$ pip install fastapi-restful-extension
```
